## Description

The Number Guessing Game is a Python-based interactive game where the computer generates a random number and the player attempts to guess it. The system provides hints such as "Too High" or "Too Low" until the correct number is guessed.

This project helps in understanding loops, conditionals, user input handling, and random number generation.



## Features

* Random number generation
* User guess input
* Too High / Too Low hints
* Attempt counter
* Interactive gameplay
* Simple command-line interface



## Technologies Used

* Python 3
* Random Module





## How to Run

1. Install Python 3.
2. Download or clone this repository.
3. Open Terminal or Command Prompt.
4. Run:

python number_guessing_game.py

5. Enter guesses until you find the correct number.
